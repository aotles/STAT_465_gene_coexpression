# STAT_465_gene_coexpression
Final project Gene Co-Expression Networks - Geoffrey Bonnanzio, Sam Ozminkowski, and Arel Otles for Stats 465 at NW University

For analysis using GWENA and WGCNA upon the mood disorder database, look within the WGCNA directory
  - GWENA paper: https://bmcbioinformatics.biomedcentral.com/articles/10.1186/s12859-021-04179-4
  - GWENA github: https://github.com/Kumquatum/GWENA
For figures produced using the original Zhang and Horvath Paper, look within the tutorial directory
  - original paper located here: https://www.degruyter.com/document/doi/10.2202/1544-6115.1128/html
For figures produced using K-means, look within the k-means directory
